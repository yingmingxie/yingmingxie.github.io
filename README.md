# XIE Lab Academic Page

Static academic/lab homepage for Ying-Ming Xie and XIE Lab at Shanghai Jiao Tong University.

## Files

- `index.html` — page content
- `styles.css` — visual design and responsive layout
- `script.js` — mobile menu and footer year
- `assets/yingming-xie.jpg` — profile photo
- `assets/CV_English.pdf` and `assets/CV_chinese.pdf` — copied CV PDFs

## Preview

Open `index.html` directly in a browser, or serve the folder with any static server.

```bash
cd academic_page
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
